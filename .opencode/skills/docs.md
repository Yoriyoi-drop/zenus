# Documentation Review Skill

This skill handles comprehensive documentation review and improvement for Zenus OS. It ensures clear, accurate, and comprehensive documentation for production use.

## Core Responsibilities
- Review documentation completeness and accuracy
- Assess documentation structure and organization
- Evaluate documentation usability for different audiences
- Identify documentation gaps and improvement opportunities
- Ensure documentation supports production operations

## Review Focus

### Documentation Coverage
- Architecture documentation (ARCHITECTURE.md, SUMMARY.md, ROADMAP.md)
- Development documentation (CONTRIBUTING.md, SECURITY.md)
- User documentation (README.md, CHANGELOG.md)
- API documentation (if applicable)
- Installation and deployment documentation

### Documentation Quality
- Clarity and accuracy
- Code examples and tutorials
- Architecture diagrams and explanations
- Best practices and conventions
- Version history and migration guides

### Documentation Structure
- Information architecture
- Audience segmentation (developers, operators, users)
- Search and navigation
- Cross-references and consistency
- Version control and updates

### Production Documentation
- Operating procedures
- Troubleshooting guides
- Monitoring and maintenance
- Security procedures
- Recovery and failover procedures
- Performance tuning guides

## Output Format
- Documentation completeness score (0-100)
- Documentation quality assessment
- Audience-specific reviews
- Structure and organization analysis
- Content accuracy verification
- Improvement recommendations
- Writing style and formatting suggestions
- Technical depth assessment

## Integration
Connects with:
- AGENTS.md for documentation standards
- Production review skill for operational documentation
- Security review skill for security documentation
- Version control history and changelog analysis
- User feedback and support tickets (if available)
