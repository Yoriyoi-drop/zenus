# Testing Review Skill

This skill handles comprehensive testing review and improvement assessment for Zenus OS. It evaluates:

## Core Responsibilities
- Assess testing coverage and effectiveness
- Review testing architecture and strategies
- Evaluate test quality and maintenance
- Identify testing gaps and improvement opportunities
- Ensure production readiness through comprehensive testing

## Review Focus

### Testing Coverage Analysis
- Unit testing coverage (current: 25 tests)
- Integration testing coverage
- System testing coverage
- Security testing coverage
- Performance testing coverage

### Testing Architecture
- Test framework and tooling
- Test organization and structure
- Test automation opportunities
- Continuous integration integration
- Test data management

### Test Quality Assessment
- Test reliability and stability
- Test maintenance overhead
- Test execution performance
- Test isolation and independence
- Test coverage quality (not just quantity)

### Production Readiness Testing
- Regression testing capabilities
- Chaos engineering support
- Load testing capabilities
- Security testing automation
- Performance benchmarking

## Output Format
- Testing readiness assessment (0-100 score)
- Coverage gap analysis
- Test quality metrics
- Automation opportunities
- CI/CD integration recommendations
- Production testing strategies

## Integration
Connects with:
- AGENTS.md for testing policies and standards
- Cargo.toml for test framework configuration
- GitHub Actions workflows (if any)
- Code coverage reports and metrics
- Performance benchmarks
